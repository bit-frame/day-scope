# Installing DayScope
To get started, clone this repository to your server
```
https://github.com/bit-frame/day-scope.git
```
Navigate to the ``DayScope`` directory and install the required packages
```
npm install
```
To start DayScope, run ``node app.js``
```
node app.js
```
